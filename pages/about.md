# **About Me**

Hey, I’m **{{ site.author.name }}** and I love messing around with tech 💻✨. I’m always exploring how tech works 🔍⚙️—whether it’s tinkering with Chrome extensions 🖱️ or learning new programming techniques 👨‍💻!
